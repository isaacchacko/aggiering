# aggier.ing

**Website:** [aggier.ing](https://aggier.ing)

[What’s a webring?](https://en.wikipedia.org/wiki/Webring)

Howdy! Welcome to an unofficial webring for students and alumni from [Texas A&M University](https://www.tamu.edu/) with personal sites/portfolios.

Want to add your site? [Open a pull request here.](https://github.com/isaacchacko/aggiering/pulls)

Hope you like it! If you have any questions or feedback, feel free to reach out.

## Join the list

Membership works through GitHub: you fork this repo, add your entry, and open a PR. Full steps, rules, and a template for your pull request description are in **[Contributing](public/CONTRIBUTING.md)**.

## Optional: links on your own site

Some people add “previous” / “next” links on their personal sites that go through the hub so visitors can move along the ring. If you’re curious, the hub is [aggier.ing](https://aggier.ing); details of how those links work live in this repo’s source if you need them for your setup.
